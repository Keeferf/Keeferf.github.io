Earthub

Duration

January to Present

Brief:

Built a Java-based elevator simulation, featuring robust system design, UML documentation, and a polished presentation to showcase key engineering decisions. This project simulates the changing of floors, elevator faults on any of the floors, and timing changes between the usages of the elevetor. This is all done through the use of schedulers and commucation of different IPs, the system also includes a graphical user interface (GUI) to show the simulation.

Responsibilities:

  • Redesigned & reworked the old Earthub website.
  • Implemented the fault reading system.
  • Developed the floor system and timing of when elevators stop.
  • Collaborated with team members to create UML diagrams and documentation.

Technologies Used:

  • Java
  • JavaScript
  • HTML/CSS
  • MongoDB
  • JSON
  • Kubernetes

Outcome:

A fully designed and developed simulation system that can be setup and used on any local machine, the system is able to operate across 3 different computers when supplied with the IP address.
