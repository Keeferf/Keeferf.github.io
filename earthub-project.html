Earthub

Duration

January to Present

Brief:

Earthub is a modern web platform designed to promote sustainable practices and environmental awareness. The project involved a complete website redesign, focusing on performance, usability, and scalability. Key improvements included optimizing search engine visibility (SEO), streamlining backend operations with MongoDB, and enhancing the overall user experience. The system was built using Java, JavaScript, and Kubernetes, ensuring a seamless and scalable architecture.

Responsibilities:

  • Redesign and redevelopment of the Earthub website, improving UI/UX and performance.
  • Optimized SEO strategies to enhance discoverability and search rankings.
  • Integrated backend services, ensuring smooth data handling with MongoDB.
  • Developed key features such as user authentication, content management, and interactive elements.

Technologies Used:

  • Java
  • JavaScript
  • HTML/CSS
  • MongoDB
  • JSON
  • Kubernetes

Outcome:

The redesigned Earthub website has significantly improved user engagement, loading speeds, and search engine rankings. The updated architecture enables smooth content management and seamless user interactions. With a scalable backend and Kubernetes-based deployment, the platform can efficiently handle increased traffic and future expansions. These improvements ensure that Earthub remains a reliable and impactful platform for promoting sustainability.
