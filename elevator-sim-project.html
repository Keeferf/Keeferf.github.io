Elevator Simulation System

Duration

January to April 2023 (4 months)

Brief:

A Java-based elevator simulation system that involves system design, communication between multiple IPs, and fault handling. The project includes UML documentation and a GUI to visually represent the simulation. Key features include floor system management, elevator timing, and fault detection, utilizing schedulers for smooth operation.

Responsibilities:

  • Designed and implemented core elevator simulation logic.
  • Developed a fault detection and handling system.
  • Implemented floor system management and elevator timing controls.
  • Contributed to UML diagrams and project documentation.

Technologies Used:

  • Java
  • UDP

Outcome:

The system is designed to run on multiple machines, allowing synchronization via IP communication. It simulates an elevator system with fault detection, offering a comprehensive and functional demonstration of key engineering concepts.
