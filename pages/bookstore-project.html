Amazin Bookstore

Duration

September to December 2024 (4 months)

Brief:

AmazinBookstore is an online bookstore management system that allows users to browse, search for books, and manage their shopping cart for purchasing. The system also provides functionality for bookstore owners to upload and edit book information.

Responsibilities:

  • Designed and implemented RESTful controllers: BookController, CartController, PurchaseController, UserController.
  • Developed and integrated user authentication and role management.
  • Implemented book recommendation features based on user purchases.
  • Optimized database interactions and deployed the system to Azure.
  • Created UML class and sequence diagrams to document system architecture.

Technologies Used:

  • Java
  • Spring Boot
  • HTML/CSS
  • JavaScript
  • MongoDB
  • Azure
  • GitHub Actions

Outcome:

The successful development of AmazinBookstore resulted in a fully functional and scalable online bookstore. Users can efficiently browse, search, and purchase books while bookstore owners can manage inventory with ease. The implementation of authentication and role management ensures a secure user experience. The book recommendation system enhances user engagement by providing personalized suggestions. With deployment on Azure and a well-structured backend, the application is capable of handling future scalability needs. The integration of GitHub Actions for CI/CD has streamlined the development and deployment process, ensuring high code quality and reliability.
